Class BlockTransformation<TInput, TOutput>

A block transformation will wait for all data to be loaded into the buffer before the transformation is applied. After all data is in the buffer, the transformation is execution and the result posted into the targets.

    inputDataAsList => {

    });
block.LinkTo(dest);
